Berikut panduan untuk menggunakan program service ini dengan Command Line Interface
1. Memastikan Kodingan Tidak Ada Error
   Penting sekali bagi kita agar memastikan bahwa kodingan program ini berjalan dengan baik
   tanpa ada error.
2. Membuka Terminal
   Untuk membuka terminal bisa langsung membukanya di Visual Studio Code dengan menggunakan
   Ctrl + Shift + `
3. Membuka Direktori File biodata.go Disimpan
   Setelah terminal dibuka, selanjutnya mengarahkan pada direktori dimana file biodata.go 
   Disimpan
4. Menjalankan File go
   Untuk menjalankan file go dengan menggunakan perintah go run biodata.go, berhubung pada 
   program ini kita menggunakan argument maka setelah mengetikkan perintah go run biodata.go
   kita harus menyisipkan argument contohnya 1, maka output dari program ini adalah akan 
   menampilkan student dengan absen no 1 beserta nama, alamat, pekerjaan, dan alasan mengikuti
   FGA Golang Kominfo x Hacktiv8 ini.
